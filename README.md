## :construction_worker: Como rodar

```bash
# Clone o repositório
git clone https://github.com/paulohdelia/linkedin-ui-clone.git
```

```bash
# Instale todas as dependências
yarn
# ou
npm install
```

```bash
# Rode o projeto
yarn start
# ou
npm start
```

Acesse **http://localhost:3000** para ver o resultado
